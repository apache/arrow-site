◆ key()

Builder* parquet::ColumnEncryptionProperties::Builder::key ( std::string  column_key)

Set a column-specific key.

If key is not set on an encrypted column, the column will be encrypted with the footer key. keyBytes Key length must be either 16, 24 or 32 bytes. The key is cloned, and will be wiped out (array values set to 0) upon completion of file writing. Caller is responsible for wiping out the input key array.
