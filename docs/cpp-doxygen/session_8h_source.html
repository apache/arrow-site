37 // the OpenSession RPC and uses it to create and return operations.
38 //
39 // Sessions are created with Service::OpenSession(). They must have Close
40 // called on them before they can be deleted.
41 //
42 // Executing RPCs with an Operation corresponding to a particular Session after
43 // that Session has been closed or deleted is undefined.
