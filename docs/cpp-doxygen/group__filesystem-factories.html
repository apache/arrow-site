Create a new FileSystem by URI.

A scheme-less URI is considered a local filesystem path. Recognized schemes are "file", "mock" and "hdfs".

Parameters
[in]uria URI-based path, ex: file:///some/local/path
[out]out_path(optional) Path inside the filesystem.
