Detailed Description

DataSourceDiscovery provides a way to inspect a DataSource potential schema before materializing it.

Thus, the user can peek the schema for data sources and decide on a unified schema. The pseudocode would look like

def get_dataset(factories): schemas = [] for f in factories: schemas.append(f.Inspect())

common_schema = UnifySchemas(schemas)

sources = [] for f in factories: f.SetSchema(common_schema) sources.append(f.Finish())

return Dataset(sources, common_schema)
