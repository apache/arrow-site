Detailed Description

A PathForest is a utility to transform a vector of FileStats into a forest representation for tree traversal purposes.

Note: there is no guarantee of a shared root. Each node in the graph wraps a FileStats. Files are expected to be found only at leaves of the tree.

Transforms a FileStats vector into a forest.

The caller should ensure that stats does not contain duplicates.

Vector(s) of associated objects (IE associated[i] is associated with stats[i]) may be passed for reordering. (After construction, associated[i] is associated with forest[i]).
