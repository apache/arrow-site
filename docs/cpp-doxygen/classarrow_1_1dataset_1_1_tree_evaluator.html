Evaluate expr against each row of a RecordBatch.

Returned Datum will be of either SCALAR or ARRAY kind. A return value of ARRAY kind will have length == batch.num_rows() An return value of SCALAR kind is equivalent to an array of the same type whose slots contain a single repeated value.

expr must be validated against the schema of batch before calling this method.
