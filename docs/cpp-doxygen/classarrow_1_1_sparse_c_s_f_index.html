Detailed Description

EXPERIMENTAL: The index data for a CSF sparse tensor.

A CSF sparse index manages the location of its non-zero values by set of prefix trees. Each path from a root to leaf forms one tensor non-zero index. CSF is implemented with three vectors.

Vectors inptr and indices contain N-1 and N buffers respectively, where N is the number of dimensions. Axis_order is a vector of integers of legth N. Indptr and indices describe the set of prefix trees. Trees traverse dimensions in order given by axis_order.
