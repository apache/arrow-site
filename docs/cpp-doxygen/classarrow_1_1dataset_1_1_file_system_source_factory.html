Build a FileSystemSourceFactory from a fs::FileSelector.

The selector will expand to a vector of FileStats. The expansion/crawling is performed in this function call. Thus, the finalized Source is working with a snapshot of the filesystem. If options.partition_base_dir is not provided, it will be overwritten with selector.base_dir.
