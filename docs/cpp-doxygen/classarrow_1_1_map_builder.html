Detailed Description

Builder class for arrays of variable-size maps.

To use this class, you must append values to the key and item array builders and use the Append function to delimit each distinct map (once the keys and items have been appended) or use the bulk API to append a sequence of offests and null maps.

Key uniqueness and ordering are not validated.
