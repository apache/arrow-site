◆ MakeNonNull()

template<typename T >
T* arrow::util::MakeNonNull ( T *  maybe_null)
inline

Returns maybe_null if not null or a non-null pointer to an arbitrary memory that shouldn't be dereferenced.

Memset/Memcpy are undefined when a nullptr is passed as an argument use this utility method to wrap locations where this could happen.

Note: Flatbuffers has UBSan warnings if a zero length vector is passed. https://github.com/google/flatbuffers/pull/5355 is trying to resolve them.
