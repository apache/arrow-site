Parameters
[in]is_adjusted_to_utcset true if the data is UTC-normalized
[in]time_unitthe resolution of the timestamp
[in]is_from_converted_typeif true, the timestamp was generated by translating a legacy converted type of TIMESTAMP_MILLIS or TIMESTAMP_MICROS. Default is false.
