Builder* parquet::ColumnDecryptionProperties::Builder::key ( const std::string &  key)

Set an explicit column key.

If applied on a file that contains key metadata for this column the metadata will be ignored, the column will be decrypted with this key. key length must be either 16, 24 or 32 bytes.
