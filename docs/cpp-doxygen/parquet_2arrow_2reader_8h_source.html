68 // TODO(wesm): nested data does not always make sense with this user
69 // interface unless you are only reading a single leaf node from a branch of
70 // a table. For example:
71 //
72 // repeated group data {
73 // optional group record {
74 // optional int32 val1;
75 // optional byte_array val2;
76 // optional bool val3;
77 // }
78 // optional int32 val4;
79 // }
80 //
81 // In the Parquet file, there are 3 leaf nodes:
82 //
83 // * data.record.val1
84 // * data.record.val2
85 // * data.record.val3
86 // * data.val4
87 //
88 // When materializing this data in an Arrow array, we would have:
89 //
90 // data: list<struct<
91 // record: struct<
92 // val1: int32,
93 // val2: string (= list<uint8>),
94 // val3: bool,
95 // >,
96 // val4: int32
97 // >>
98 //
99 // However, in the Parquet format, each leaf node has its own repetition and
100 // definition levels describing the structure of the intermediate nodes in
101 // this array structure. Thus, we will need to scan the leaf data for a group
102 // of leaf nodes part of the same type tree to create a single result Arrow
103 // nested array structure.
104 //
105 // This is additionally complicated "chunky" repeated fields or very large byte
106 // arrays

250 // At this point, the column reader is a stream iterator. It only knows how to
251 // read the next batch of values for a particular column from the file until it
252 // runs out.
253 //
254 // We also do not expose any internal Parquet details, such as row groups. This
255 // might change in the future.
256 class PARQUET_EXPORT ColumnReader {
257  public:
258  virtual ~ColumnReader() = default;
259 
260  // Scan the next array of the indicated size. The actual size of the
261  // returned array may be less than the passed size depending how much data is
262  // available in the file.
263  //
264  // When all the data in the file has been exhausted, the result is set to
265  // nullptr.
266  //
267  // Returns Status::OK on a successful read, including if you have exhausted
268  // the data available in the file.
269  virtual ::arrow::Status NextBatch(int64_t batch_size,
270  std::shared_ptr<::arrow::ChunkedArray>* out) = 0;
271 };
