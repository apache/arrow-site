Detailed Description

Simple arena allocator.

Memory is allocated from system in units of chunk-size, and dished out in the requested sizes. If the requested size > chunk-size, allocate directly from the system.

The allocated memory gets released only when the arena is destroyed, or on Reset.

This code is not multi-thread safe, and avoids all locking for efficiency.
