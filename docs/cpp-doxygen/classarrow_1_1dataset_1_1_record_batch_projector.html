Detailed Description

Project a RecordBatch to a given schema.

Projected record batches will reorder columns from input record batches when possible, otherwise the given schema will be satisfied by augmenting with null or constant columns.

RecordBatchProjector is most efficient when projecting record batches with a consistent schema (for example batches from a table), but it can project record batches having any schema.
