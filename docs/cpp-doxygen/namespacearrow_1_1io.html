int64_t  block_size

Return an iterator on an input stream.

The iterator yields a fixed-size block on each Next() call, except the last block in the stream which may be smaller. Once the end of stream is reached, Next() returns nullptr (unlike InputStream::Read() which returns an empty buffer).
