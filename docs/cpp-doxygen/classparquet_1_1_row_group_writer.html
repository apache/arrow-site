◆ NextColumn()

ColumnWriter* parquet::RowGroupWriter::NextColumn ( )

Construct a ColumnWriter for the indicated row group-relative column.

To be used only with ParquetFileWriter::AppendRowGroup Ownership is solely within the RowGroupWriter. The ColumnWriter is only valid until the next call to NextColumn or Close. As the contents are directly written to the sink, once a new column is started, the contents of the previous one cannot be modified anymore.
