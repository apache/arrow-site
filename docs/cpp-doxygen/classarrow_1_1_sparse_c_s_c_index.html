Detailed Description

EXPERIMENTAL: The index data for a CSC sparse matrix.

A CSC sparse index manages the location of its non-zero values by two vectors.

The first vector, called indptr, represents the range of the column; the i-th column spans from indptr[i] to indptr[i+1] in the corresponding value vector. So the length of an indptr vector is the number of columns + 1.

The other vector, called indices, represents the row indices of the corresponding non-zero values. So the length of an indices vector is same as the number of non-zero-values.
