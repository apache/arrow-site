Arrays and Scalars¶

Factory Functions¶

These functions create new Arrow arrays:

array(obj[, type, mask, size, from_pandas])

Create pyarrow.Array instance from a Python object.

nulls(size[, type])

Create a strongly-typed Array instance with all elements null.
