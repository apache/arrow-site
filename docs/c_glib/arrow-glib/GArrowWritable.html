Prerequisites

GArrowWritable requires GObject.

Known Implementations

GArrowWritable is implemented by GArrowBufferOutputStream, GArrowFileOutputStream, GArrowGIOOutputStream and GArrowOutputStream.

Description

GArrowWritable is an interface for output. Output must be writable.
