# Cast options

Cast options

## Usage

``` r
cast_options(safe = TRUE, ...)
```

## Arguments

- safe:

  logical: enforce safe conversion? Default `TRUE`

- ...:

  additional cast options, such as `allow_int_overflow`,
  `allow_time_truncate`, and `allow_float_truncate`, which are set to
  `!safe` by default

## Value

A list
