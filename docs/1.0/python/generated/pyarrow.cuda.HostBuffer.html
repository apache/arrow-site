pyarrow.cuda.HostBuffer¶

class pyarrow.cuda.HostBuffer¶

Bases: pyarrow.lib.Buffer

Device-accessible CPU memory created using cudaHostAlloc.

To create a HostBuffer instance, use

cuda.new_host_buffer(<nbytes>)
