# Create a Buffer

<div class="ref-description section level2">

Create a Buffer

</div>

<div class="section level2">

## Usage

<div class="sourceCode">

``` r
buffer(x)
```

</div>

</div>

<div class="section level2">

## Arguments

-   x:

    R object. Only raw, numeric and integer vectors are currently
    supported

</div>

<div class="section level2">

## Value

an instance of `Buffer` that borrows memory from `x`

</div>
