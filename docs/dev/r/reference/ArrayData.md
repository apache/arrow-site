# ArrayData class

<div class="ref-description section level2">

The `ArrayData` class allows you to get and inspect the data inside an
`arrow::Array`.

</div>

<div class="section level2">

## Usage

<div class="sourceCode">

    data <- Array$create(x)$data()

    data$type
    data$length
    data$null_count
    data$offset
    data$buffers

</div>
