# ParquetReaderProperties class

This class holds settings to control how a Parquet file is read by
[ParquetFileReader](https://arrow.apache.org/docs/r/reference/ParquetFileReader.md).

## Factory

The `ParquetReaderProperties$create()` factory method instantiates the
object and takes no arguments.

## Methods

- `$thrift_string_size_limit()`

- `$set_thrift_string_size_limit()`

- `$thrift_container_size_limit()`

- `$set_thrift_container_size_limit()`
