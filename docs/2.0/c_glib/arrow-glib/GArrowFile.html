Prerequisites

GArrowFile requires GObject.

Known Implementations

GArrowFile is implemented by GArrowBufferInputStream, GArrowBufferOutputStream, GArrowCUDABufferInputStream, GArrowCUDABufferOutputStream, GArrowCompressedInputStream, GArrowCompressedOutputStream, GArrowFileOutputStream, GArrowGIOInputStream, GArrowGIOOutputStream, GArrowInputStream, GArrowMemoryMappedInputStream, GArrowOutputStream and GArrowSeekableInputStream.
