# Take an object of length 1 and repeat it.

Take an object of length 1 and repeat it.

## Usage

``` r
repeat_value_as_array(object, n)
```

## Arguments

- object:

  Object of length 1 to be repeated - vector, `Scalar`, `Array`, or
  `ChunkedArray`

- n:

  Number of repetitions

## Value

`Array` of length `n`
