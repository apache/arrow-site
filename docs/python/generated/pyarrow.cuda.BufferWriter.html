pyarrow.cuda.BufferWriter¶

class pyarrow.cuda.BufferWriter¶

Bases: pyarrow.lib.NativeFile

File interface for writing to CUDA buffers.

By default writes are unbuffered. Use set_buffer_size to enable buffering.
