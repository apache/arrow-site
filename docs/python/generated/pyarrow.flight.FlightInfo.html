pyarrow.flight.FlightInfo¶

class pyarrow.flight.FlightInfo¶

Bases: object

A description of a Flight stream.

__init__()¶

Create a FlightInfo object from a schema, descriptor, and endpoints.

Parameters

  • schema (Schema) – the schema of the data in this flight.

  • descriptor (FlightDescriptor) – the descriptor for this flight.

  • endpoints (list of FlightEndpoint) – a list of endpoints where this flight is available.

  • total_records (int) – the total records in this flight, or -1 if unknown

  • total_bytes (int) – the total bytes in this flight, or -1 if unknown
