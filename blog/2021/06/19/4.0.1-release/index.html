Apache Arrow 4.0.1 Release

Published 19 Jun 2021
By The Apache Arrow PMC (pmc)

The Apache Arrow team is pleased to announce the 4.0.1 release. This release covers general bug fixes on the different implementations, notably C++, R, Python and JavaScript. The list is available here, with the list of contributors here and changelog here.
